# Como baixar o projeto

```bash

    # Clonar o repositório
    $ git clone https://github.com/Tiaguin061/siteResponsivo
    # Entrar no diretório
    $ cd siteResponsivo

    # Instalar as dependências
    $ yarn install

    # Iniciar o Projeto
    $ yarn start

```
